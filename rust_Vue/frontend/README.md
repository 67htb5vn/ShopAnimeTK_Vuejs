# Shop Anime Admin Frontend - Vue 3

## Chạy frontend

```bash
npm install
npm run dev
```

Frontend chạy tại `http://localhost:5173`, proxy `/api` sang backend Rust `http://127.0.0.1:8080`.

## Trang đã có

- Dashboard
- Quản lý sản phẩm
- Quản lý danh mục hãng
- Quản lý anime/hoạt hình
- Quản lý hóa đơn + cập nhật trạng thái
- Quản lý người dùng
- Quản lý khuyến mãi
- Quản lý tag
